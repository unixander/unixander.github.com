Abrakadabra_Chrome

Release Date: May 2012
Current version: 1.1.0.3
Current test version: 1.1.0.5
Current status: developing
My role: developer
Website: [link]
Description: Chrome extension for Russian users. It helps to correct words written in another keyboard layout. For example, Russian word "Привет" written in English layout "Ghbdtn". At the moment, available two directions of the correction: Eng->Rus, Rus->Eng. There is both automatic correction or correction on hotkey pressed.
Supports: vk.com, facebook.com, Gmail, imo.im chats.
Developer's notes: Now available only for Chrome browser. The purpose was to practice in pure javascript and to write my first extension for Chrome.
